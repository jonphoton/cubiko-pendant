Install Cubiko Pendant

Steps:

  1. Unplug the M5Dial from the Cubiko.
  2. Plug it into your laptop's USB-C port while holding the G0 side button. Release G0 after it's plugged in.
  3. Click Install below and choose the serial port that appeared (usually /dev/tty.usbmodem… on macOS or COMx on Windows).
  4. Wait ~30 seconds. When it says "Installation complete", unplug and plug back into the Cubiko.
Your browser does not support WebSerial. Please use Chrome or Edge on a laptop. This page must be served over HTTPS. If you're viewing this locally, try the hosted version at jonphoton.github.io/cubiko-pendant.
Only do this the first time. The web installer above wipes the pendant's flash, which means you'll have to redo WiFi setup after each use. For every subsequent firmware update, use the built-in Update firmware link at the bottom of http://cubiko.local instead — it does an over-the-air update that preserves your saved WiFi credentials and won't brick the device if it fails.

First-boot WiFi

After install, the pendant creates a WiFi network called M5DialCubiko-Setup. Join it from your phone; a captive portal lets you pick your home WiFi. See the README for details.

Uploading G-code

Once the pendant has joined your WiFi, open http://cubiko.local/ from any device on the same network — laptop, phone, whatever — and drag G-code files onto the page. The pendant serves its own drag-and-drop UI, so there's nothing to install on the computer.
